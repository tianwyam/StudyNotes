2.2、安装服务

my.ini文件配置好后就可以在cmd中安装mysqld服务了，在cmd中运行命令

命令：

mysqld --install MySQL   --defaults-file="D:\Program Files(x86)\MySql\my.ini"

其中的MySQL是安装服务器的名称，可以指定任何名称。安装完成后会提示下面的信息：Service successfully installed，即表示安装成功。
